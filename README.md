# 3D Data Scientist Portfolio

**Live Portfolio**: [https://my-portfolio-jacx.onrender.com](https://my-portfolio-jacx.onrender.com)

A premium, highly interactive 3D portfolio website custom-built for an Entry-Level Data Scientist. The design utilizes a modern dark theme with vibrant cybernetic accents, cutting-edge CSS glassmorphism, and high-performance WebGL animations.

## 🌟 Key Features

- **3D Cosmic Galaxy Background**: A mesmerizing, interactive Three.js galaxy simulation featuring flowing spiral arms and high-speed shooting stars that dynamically react to your mouse movements.
- **Interactive Skills Sphere**: A fully 3D rotating TagCloud displaying data science skills, paired side-by-side with sleek, neon-glowing progress bars.
- **Dynamic Scroll Animations**: Powered by GSAP (GreenSock), portfolio elements gracefully fade, slide, and stagger into view as you scroll down the page.
- **Premium UI Enhancements**:
  - **About Section**: Split layout featuring a continuously rotating 3D glowing ring visual.
  - **Projects Section**: 3D tilt-effect cards that reveal vibrant, glowing gradient borders on hover.
  - **Education Section**: A custom timeline featuring animated, pulsating neon dots.
  - **Contact Section**: Wide, floating interactive badges with premium glowing hover states.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices seamlessly.

## 🛠️ Technologies Used

- **HTML5 & Vanilla CSS3**: For structure, layout, custom CSS variables, keyframe animations, and glassmorphism styling.
- **JavaScript (ES6+)**: For interaction, DOM manipulation, and library execution.
- **Three.js**: For the complex 3D galaxy particle simulation.
- **GSAP & ScrollTrigger**: For high-performance, scroll-driven reveal animations.
- **TagCloud.js**: For the interactive 3D rotating skills sphere.
- **Vanilla-Tilt.js**: For the 3D interactive physics on project cards.
- **FontAwesome**: For crisp, scalable vector icons.
- **Google Fonts**: Utilizes the modern *Inter* and *Outfit* font families.

## 🚀 How to Run Locally

Since this project uses external libraries via CDN and basic HTML/CSS/JS, no complex build process is required. You can serve it using any simple local web server. 

If you have Python installed, open your terminal (Command Prompt/PowerShell) in the `d:\Portfolio` directory and run:

```bash
python -m http.server 8000
```

Then, open your web browser and navigate to `http://localhost:8000` to view the portfolio in action.
